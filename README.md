## Quick Start
```bash
npx clawcompany
```

2 steps. 30 seconds. No Docker. No config files. No proxy.

> Requires Node.js 20+. Supports ClawAPI, Anthropic, OpenAI, Google, Ollama.

## What is ClawCompany?

You are the Chairman. Your AI team executes autonomously.
You → "Analyze NVDA for 2026"
↓
Leader decomposes into work streams
→ Researcher gathers data (web search, price feed)
→ Analyst builds the model (code interpreter)
→ Writer formats the report
↓
You: read the report. $0.06. Done.

**The right model for the right job.** Professional work gets Opus/Sonnet. Routine work gets Flash-Lite. 30× cheaper than all-Opus.

→ [See how it works](https://clawcompany.org#how)

## 🧠 4-Layer Company Memory

| Layer | What it does |
|-------|-------------|
| **Layer 4 — Chairman** | Your preferences, language, industry. Always injected. |
| **Layer 3 — Company** | 4 partitions: Culture, Decisions, Learnings, Tech Stack. Auto-categorized. LLM-compressed. |
| **Layer 2 — Archive** | Compressed originals preserved. Searchable. Nothing is ever lost. |
| **Layer 1 — Session** | Current conversation context. |

~400 tokens injected per mission. OpenClaw injects 5,000+. Same quality, 1/13 the cost.

→ [See memory architecture](https://clawcompany.org#memory)
